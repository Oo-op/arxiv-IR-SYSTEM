# 信息检索系统实验报告提纲

## 1. 实验目标

- 设计并实现一个基于向量空间模型的信息检索系统
- 完成文档抓取、预处理、倒排索引构建、TF-IDF 检索与结果评价

## 2. 数据来源与主题

- 数据源：arXiv
- 主题：`cs.AI`
- 文档数量：`120`
- 文档形式：论文标题与摘要

## 3. 系统实现过程

### 3.1 数据采集

- 通过爬取 arXiv 页面获取论文标题、摘要、URL、日期、作者等字段

### 3.2 文本预处理

- 去除 HTML 标签
- 文本规范化
- 小写化
- 正则分词

### 3.3 词表与倒排索引

- 构建词项词典
- 统计词频与文档频率
- 建立 posting list

### 3.4 检索模型

- TF 计算
- IDF 计算
- 文档向量与查询向量构建
- 余弦相似度排序

### 3.5 结果展示

- 输出相关度、标题、摘要片段、URL、日期

### 3.6 人工评价

- Precision@5
- Precision@10

## 4. 实验结果分析

- 哪些查询结果较好
- 哪些查询容易出现误检
- 原因分析：摘要短、词汇歧义、无停用词处理等

## 5. 环境与社会影响讨论

### 5.1 正面影响

- 提高学术信息获取效率
- 帮助用户快速定位研究主题

### 5.2 潜在问题

- 结果偏差与相关性误判
- 学术摘要不等于全文内容
- 检索排序可能放大热门术语偏置

## 6. 不足与改进方向

- 当前仅使用 TF-IDF，没有语义检索
- 没有 Web 界面
- 没有停用词与词干提取
- 可进一步扩展为 BM25 或向量检索
